Easy Installation

FREE Version Installation

  1. Download the FREE version
  2. Go to WordPress Admin → Plugins → Add New
  3. Click "Upload Plugin" at the top
  4. Select the ZIP file and click "Install Now"
  5. Click "Activate Plugin"

PRO Version Installation

  1. Purchase a PRO license
  2. Download PRO version from your account
  3. Install and activate like the FREE version
  4. Enter your license key in Settings → License
  5. PRO features activate automatically

Initial Setup

  1. Go to Email Domain Restriction in the admin menu
  2. Click "Domain Whitelist" to add approved domains
  3. Configure settings in the Settings tab
  4. Test with whitelisted and non-whitelisted emails
  5. (PRO) Configure integrations and advanced features

Frequently Asked Questions

What's the difference between FREE and PRO?

The FREE version provides domain whitelisting, basic analytics, and registration logging. PRO adds advanced analytics with Chart.js visualizations, email validation APIs (ZeroBounce, Kickbox, Hunter.io, NeverBounce), role-based mapping, WooCommerce & BuddyPress integrations, geolocation, webhooks, and more. See the pricing section for full comparison.

Which email validation service should I use?

All four services (ZeroBounce, Kickbox, Hunter.io, NeverBounce) are excellent. ZeroBounce offers the most comprehensive validation, Kickbox is fast and reliable, Hunter.io is great for B2B, and NeverBounce has competitive pricing. You can test connections to all services and choose your preferred one.

Can I use wildcards for subdomains?

Yes! Use *.example.com to allow all subdomains like user@sub.example.com, user@another.example.com, etc. Works in both FREE and PRO versions.

Does this work with WooCommerce and BuddyPress?

Yes! PRO version includes full WooCommerce integration (checkout + My Account validation, B2B mode, role assignment) and BuddyPress integration (registration validation, group-level restrictions, member type assignment).

How does role-based mapping work?

PRO feature: You can map email domains to WordPress roles. For example, @company.com → Editor, *.partners.com → Contributor. Supports wildcards, priorities, and automatic conflict resolution when multiple patterns match.

Can I block registrations from specific countries?

Yes! PRO version includes geolocation with allow/block modes. You can create a whitelist of allowed countries or a blacklist of blocked countries. Uses IP address detection with caching for performance.

How does rate limiting work?

PRO feature: Prevents spam by limiting registration attempts per domain and per IP address. Configure time windows (e.g., 1 hour) and maximum attempts (e.g., 5 attempts). Automatic temporary blocking when limits exceeded.

What are webhooks used for?

PRO feature: Get real-time notifications when registrations are allowed or blocked. Send to your own endpoints or integrate with services like Slack, Discord, Zapier. Includes HMAC signature verification for security.

Can I export analytics data?

Yes! PRO version lets you export analytics to CSV or PDF. Also includes scheduled email reports (daily/weekly/monthly) sent automatically to specified addresses.

Does this work on multisite?

Yes, it works on both single-site and multisite WordPress installations. Each site requires its own license for PRO features.

Do I get updates and support?
